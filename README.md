# ip-lookup
IP lookup web application using Google Maps API and ipinfo.io

Displays:
- IP
- Hostname
- City
- Region
- Country
- ORG (if applicable)

[Demo](musa11971.github.io/ip-lookup)

To run this yourself, replace *[your_api_key]* in index.html with your Google Maps API key.
